# Policy Feedback Sentiment Analysis

A comprehensive web application for analyzing public sentiment towards policy proposals using machine learning and natural language processing.

## Features

- **Single Text Analysis**: Analyze sentiment of individual policy feedback
- **Batch File Analysis**: Process CSV/Excel files containing multiple feedback entries
- **Interactive Dashboard**: Visualize sentiment distribution with charts and word clouds
- **Real-time Analysis**: Get instant sentiment results
- **Export Results**: Download analyzed data for further processing
